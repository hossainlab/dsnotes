String Operators¶

The + Operator¶

The + operator concatenates strings. It returns a string consisting of the operands joined together.

s1 = "Ub" 
s2 = "er"
s1+s2

'Uber'

s3 = "Fizz"
s4 = "Buzz"
s3+s4

'FizzBuzz'

The * Operator¶

The * operator creates multiple copies of a string. If s is a string and n is an integer, either of the following expressions returns a string consisting of n concatenated copies of s

  • \(s \times n\)

  • \(n \times s\)

s = "Fizz"
n = 3 
# n * s 
s * n 

'FizzFizzFizz'

# s * n 
n * s 

'FizzFizzFizz'

Note

The multiplier operand n must be an integer. You’d think it would be required to be a positive integer, but amusingly, it can be zero or negative, in which case the result is an empty string.

"foo" * -8

The in Operator¶

Python also provides a membership operator that can be used with strings. The in operator returns True if the first operand is contained within the second, and False otherwise

text = "I love Bangladesh!"
# membership check 
"love" in text 

True

# Dhaka in text? 
"Dhaka" in text

False

# Dhaka not in text? 
"Dhaka" not in text

True

Built-in String Function¶

ord(c)¶

Returns an integer of the given character.

# ASCII value of a 
ord('a')

97

# ASCII value of A 
ord('A')

65

# ASCII value of # 
ord("#")

35

chr(i)¶

Converts an integer to character

# Convert integer to character 97 == a 
chr(97)

'a'

# Convert integer to character 65 == A 
chr(65)

'A'

len(s)¶

Returns the length of a string

# length of a string 
S = "Amar Sonar Bangla!"
len(S) 

18

str(param)¶

Returns a string representation of a string

# 10 into '10'
str(10)

'10'

# 10.5 into '10.5'
str(10.5)

'10.5'

String Indexing¶

In Python, strings are ordered sequences of character data, and thus can be indexed in this way. Individual characters in a string can be accessed by specifying the string name followed by a number in square brackets ([]). String indexing in Python is zero-based: the first character in the string has index 0, the next has index 1, and so on. The index of the last character will be the length of the string minus one.

# forward indexing
country = "Bangladesh"

# check length
len(country)

String Operation Function¶

s.lower()¶

Returns all characters in lowercase.

name = "Rahim"
name.lower() 

'rahim'

s.upper()¶

Returns all characters in uppercase

text = "fizzbuzz"
text.upper() 

'FIZZBUZZ'

s.capitalize()¶

Converts first letter into uppercase

text = "fizzbuzz"
text.capitalize() 

'Fizzbuzz'

s.title()¶

Convert first letter lowercase to uppercase in a sentence

text2 = "i love bangladesh!"
text2.title() 

'I Love Bangladesh!'

s.split(t)¶

Converts a sentence into List

# Split a sentence into list 
sent = "Dhaka is the capital of Bangladesh!"
sent.split(" ")

['Dhaka', 'is', 'the', 'capital', 'of', 'Bangladesh!']

s.strip()¶

Removes whitespace from text(both side)

text4 = "     FooBazar"
text4.strip() 

'FooBazar'

s.rstrip()¶

Removes whitespace from right end.

text5 = "  FooBazar        "
text5.rstrip() 

'  FooBazar'

s.lstrip()¶

Removes whitespace from left end

text6 = "  FooBazar        "
text6.lstrip()

'FooBazar        '

s.find(t)¶

Returns substring position.if the string not found return -1

text7 = "Datastics Lab"
text7.find("s")

4

text7 = "Datastics Lab"
text7.find("Lab")

10

text7 = "Datastics Lab"
text7.find("foo")

-1

s.count(t)¶

Returns the number of character in a sentence or word

country = "Japan"
country.count("a")

2

s.replace(u, v)¶

Replace value in a sequence of character(string)

text8 = "Datastics lab"
text8.replace("l", "L") 

'Datastics Lab'
